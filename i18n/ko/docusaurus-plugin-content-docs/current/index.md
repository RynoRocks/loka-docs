# Loka에 오신 것을 환영합니다

Loka는 자신만의 방식으로 교육 비즈니스를 운영하고자 하는 언어 교육자를 위해 만들어진 학습 관리 시스템입니다. 1:1 회화 수업, 그룹 수업 또는 그 사이 어떤 형태의 수업을 진행하든, Loka는 학생 관리, 수업 진행, 결제 처리를 하나의 플랫폼에서 할 수 있는 도구를 제공합니다.

## Loka의 기능

- **Living Canvas** -- 모든 수업을 살아있는 문서로 만드는 실시간 대화형 화이트보드입니다. 수업 중 교정, 어휘, 노트를 기록하고 학습자에게 즉시 공유할 수 있습니다.
- **Living Textbook** -- 교정 내용과 수업 중 주요 포인트가 자동으로 각 학습자 맞춤형 교재로 만들어집니다. 더 이상 노트를 잃어버리거나 피드백을 잊어버릴 걱정이 없습니다.
- **스케줄 및 예약** -- 학습자가 설정된 가능 시간에 직접 예약할 수 있습니다. 캘린더 연동으로 모든 일정이 자동으로 정리됩니다.
- **결제 및 구독** -- Stripe 기반 결제 시스템이 수업료, 좌석 구독, 크레딧 팩을 자동으로 처리합니다.
- **AI 연동** -- 직접 보유한 API 키(Anthropic Claude 또는 OpenAI)를 연결하여 AI 기반 수업 준비, 실시간 제안, 수업 후 피드백 기능을 활용할 수 있습니다.
- **분석 및 진도 추적** -- 학습자 진도, 수업 이력, 교수 패턴을 시간별로 추적할 수 있습니다.

## 누구를 위한 서비스인가요?

Loka는 세 가지 유형의 사용자에게 각각 맞춤형 문서를 제공합니다:

| 사용자 유형 | 설명 | 가이드 |
|-----------|------|-------|
| **교육자** | 수업을 진행하고 학습자를 관리하는 언어 교사 | [교육자 가이드](./educators/dashboard) |
| **학습자** | 수업을 예약하고, 세션에 참여하며, 진도를 확인하는 학생 | [학습자 가이드](./learners/booking) |
| **Loka Creator** | 자신만의 브랜드 교육 비즈니스를 구축하고 확장하려는 교육자 | [Creator 가이드](./creators/overview) |
