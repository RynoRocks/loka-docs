# 교육자 및 학습자 관리

Loka Creator로서 학교의 구성원, 즉 수업을 진행하는 교육자와 학습하는 학습자를 관리합니다.

## 교육자 추가

학교의 각 교육자에게는 **교육자 좌석**($6/월)이 필요합니다. Loka Creator 플랜에는 기본으로 교육자 좌석 1개가 포함됩니다.

새 교육자를 추가하려면:

1. **관리 패널 > 교육자**로 이동합니다
2. **교육자 추가**를 클릭합니다
3. 이름, 이메일 주소 및 초기 설정을 입력합니다
4. 교육자에게 계정 설정 링크가 포함된 초대 이메일이 발송됩니다

활성화되면 교육자는 다음에 접근할 수 있는 자체 대시보드를 받습니다:

- 수업 스케줄링 및 Living Textbook
- AI 기반 수업 도구 (ARP 생성, 평가, 피드백)
- 배정된 학습자의 학생 관리
- Canvas 및 화이트보드 도구

### 교육자 제거

교육자를 제거하려면:

1. **관리 패널 > 교육자**로 이동합니다
2. 교육자 이름을 클릭합니다
3. **교육자 제거**를 선택합니다
4. 제거를 확인합니다

교육자를 제거하면 현재 결제 기간 종료 시 좌석 구독이 취소됩니다. 수업 이력과 자료는 보존됩니다.

## 학습자 계정 관리

학습자는 초대 또는 가입 링크를 통해 학교에 참여합니다.

### 학습자 초대

1. **관리 패널 > 학습자**로 이동합니다
2. **학습자 추가** 또는 **초대 발송**을 클릭합니다
3. 학습자의 이메일 주소를 입력합니다
4. 학습자에게 계정 생성 안내가 포함된 이메일이 발송됩니다

### 학습자 좌석

각 활성 학습자에게는 **학습자 좌석**($6/월)이 필요합니다. 교육자 좌석과 달리, 학습자 좌석은 학습자가 수업료를 결제할 때 자동으로 갱신됩니다. 학습자가 수업 예약을 중단하면 좌석 구독이 갱신되지 않습니다.

따라서 학습자 좌석을 수동으로 취소할 필요가 없습니다 -- 학습자가 비활성화되면 자연스럽게 종료됩니다.

### 학습자 상세 정보 확인

학습자 관리 페이지에서 다음을 확인할 수 있습니다:

- 프로필 정보 및 연락처
- 수업 이력 및 예정 예약
- 진도 데이터 및 현재 레벨
- 좌석 구독 상태
- 크레딧 잔액

## 사용자 상태

Loka의 사용자는 다음 상태를 가질 수 있습니다:

| 상태 | 의미 |
|------|------|
| **활성** | 활성 계정과 활성 좌석 구독이 있는 사용자 |
| **대기** | 초대되었지만 아직 가입을 완료하지 않은 사용자 |
| **비활성** | 좌석 구독이 만료되거나 취소된 사용자 |
| **정지** | Creator가 일시적으로 비활성화한 계정 |

사용자 목록을 상태별로 필터링하여 필요한 사용자를 빠르게 찾을 수 있습니다.

## 인증

새 사용자는 인증 과정을 거칩니다:

1. **이메일 인증** -- 환영 이메일의 링크를 클릭하여 이메일 주소를 확인합니다
2. **프로필 완성** -- 필수 프로필 필드(이름, 시간대 등)를 입력합니다
3. **좌석 활성화** -- 사용자의 좌석 구독이 활성화됩니다 (Creator가 직접 또는 결제를 통해)

모든 단계가 완료되면 사용자 상태가 **활성**으로 변경되며 플랫폼을 완전히 사용할 수 있습니다.

:::tip
원래 초대를 받지 못했거나 링크가 만료된 경우 사용자의 프로필 페이지에서 초대 이메일을 재발송할 수 있습니다.
:::
