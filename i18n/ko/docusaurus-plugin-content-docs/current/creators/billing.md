# 결제 및 구독

Loka는 좌석 기반 구독 모델을 사용합니다. 학교를 위한 Creator 플랜을 선택한 후 학교가 성장함에 따라 교육자 및 학습자 좌석을 추가합니다.

## Loka Creator 플랜

| 플랜 | 가격 | 포함 내용 |
|------|------|----------|
| **Loka Creator DIY** | $6/월 | 교육자 좌석 1개 + 학습자 테스트 좌석 1개. 셀프 서비스 설정 -- 모든 것을 직접 구성합니다. |
| **Loka Creator DFY** | $29/월 | DIY의 모든 기능 + Done-For-You 플랫폼 설정 + 월 2시간 기술 지원 |
| **Loka Creator Aspire** | $199/월 | DFY의 모든 기능 + 교육 비즈니스 성장을 위한 월 2시간 파운더 코칭 |

모든 플랜에는 Loka 플랫폼의 전체 기능이 포함됩니다: Living Textbook, AI 도구, 예약 시스템, 진도 추적, 연동.

### 14-Day Challenge

Loka가 적합한지 확실하지 않다면, **14-Day Challenge**($39 일회성)는 플랫폼의 가이드 소개입니다. 14일 동안 체계적인 안내와 함께 Loka를 탐색한 후 월간 플랜으로 업그레이드할 수 있습니다.

## 교육자 좌석

학교의 각 교육자에게는 자체 좌석이 필요합니다.

- **비용:** 교육자 좌석당 $6/월
- **결제:** 표준 월간 Stripe 구독
- **포함:** Creator 플랜에 교육자 좌석 1개 포함
- **추가:** **관리 패널 > 결제 > 교육자 좌석 추가**에서 추가 좌석 구매
- **취소:** 수동 -- 더 이상 필요하지 않을 때 교육자 좌석을 취소합니다

## 학습자 좌석

각 활성 학습자에게도 좌석이 필요하지만, 학습자 결제 방식은 다릅니다.

- **비용:** 학습자 좌석당 $6/월
- **결제:** 학습자가 수업료를 결제할 때 자동 갱신
- **자동 취소:** 학습자가 수업료 결제를 중단하면 좌석이 갱신되지 않음
- **수동 관리 불필요:** 학습자 좌석은 수업 결제에 따라 자동으로 활성화 및 비활성화

이 모델은 비활성 학습자에 대해 비용을 지불하지 않음을 의미합니다. 좌석은 활성 학생 기반에 따라 자연스럽게 조정됩니다.

## Stripe 결제 포털

모든 결제는 Stripe를 통해 처리됩니다. **관리 패널 > 결제 > 결제 관리**에서 Stripe 결제 포털에 접근할 수 있습니다.

포털에서 다음을 수행할 수 있습니다:

- 청구서 확인 및 다운로드
- 결제 수단 업데이트 (신용카드 등)
- 현재 구독 및 갱신 날짜 확인
- 결제 이력 확인

## 구독 관리

### 플랜 업그레이드

상위 Creator 플랜으로 이동하려면:

1. **관리 패널 > 결제**로 이동합니다
2. **플랜 변경**을 클릭합니다
3. 새 플랜을 선택합니다
4. 변경이 즉시 적용됩니다. 현재 결제 기간의 나머지에 대해 비례 차액이 청구됩니다.

### 플랜 다운그레이드

다운그레이드는 현재 결제 기간 종료 시 적용됩니다. 포함 기능이 적은 플랜으로 다운그레이드하는 경우, 전환 전에 설정이 호환되는지 확인하세요.

### 플랜 취소

Loka Creator 구독을 취소하려면:

1. **관리 패널 > 결제**로 이동합니다
2. **구독 취소**를 클릭합니다
3. 현재 결제 기간 종료까지 접근이 유지됩니다
4. 취소 후 다시 돌아올 경우를 대비하여 데이터가 90일간 보존됩니다

:::caution
Creator 플랜을 취소하면 학교에 연결된 모든 교육자 및 학습자 좌석도 취소됩니다. 보존 기간이 종료되기 전에 필요한 데이터를 내보내세요.
:::

### 좌석 추가 또는 제거

- **좌석 추가:** **관리 패널 > 결제 > 교육자 좌석 추가** (학습자를 추가하면 결제 시 좌석이 자동 활성화됩니다)
- **교육자 좌석 제거:** 교육자 프로필로 이동하여 **교육자 제거**를 클릭합니다. 좌석은 결제 기간 종료 시 취소됩니다.
- **학습자 좌석 제거:** 학습자가 수업료 결제를 중단하면 좌석이 자동으로 취소됩니다. 수동 조치가 필요 없습니다.
