# 기술 FAQ

Loka 플랫폼 사용에 대해 자주 묻는 기술 질문입니다.

---

### 어떤 브라우저가 지원되나요?

Loka는 다음 브라우저의 최신 버전에서 가장 잘 작동합니다:

- **Google Chrome** (권장)
- **Mozilla Firefox**
- **Microsoft Edge**
- **Safari** (macOS 및 iOS)

최적의 경험을 위해 브라우저를 최신 버전으로 유지하는 것을 권장합니다. Internet Explorer는 지원되지 않습니다.

---

### 수업 시간이 잘못된 시간대로 표시됩니다. 어떻게 수정하나요?

**계정 설정 > 시간대**로 이동하여 올바른 시간대가 선택되어 있는지 확인하세요. Loka는 가입 시 시간대를 자동으로 감지하지만, 이사했거나 시스템 시계가 잘못 설정된 경우 업데이트가 필요할 수 있습니다.

모든 수업 시간, 예약 슬롯, 알림은 설정된 시간대를 사용합니다. 업데이트 후 페이지를 새로고침하면 수정된 시간이 표시됩니다.

---

### 어떤 화상 회의 플랫폼을 지원하나요?

Loka는 온라인 수업을 위해 **Zoom**과 연동됩니다. Zoom이 연결되면 온라인 수업 예약 시 Zoom 미팅 링크가 자동으로 생성되어 확인 이메일에 포함됩니다.

학교에서 다른 화상 플랫폼(Google Meet, Microsoft Teams 등)을 사용하는 경우, 교육자가 수업 세부 정보에 미팅 링크를 수동으로 추가할 수 있습니다.

---

### AI API 키를 어떻게 설정하나요?

Loka의 AI 기능(ARP 생성, 수업 평가, AI 기반 피드백)은 **Anthropic (Claude)** 또는 **OpenAI (GPT)**의 API 키가 필요합니다.

키를 설정하려면:

1. **프로필 > 설정 > AI 연동**으로 이동합니다
2. Anthropic, OpenAI 또는 둘 다의 API 키를 입력합니다
3. **선호 제공업체**를 선택합니다 (시스템이 선호 제공업체를 먼저 시도하고 필요 시 다른 제공업체로 폴백합니다)
4. **저장**을 클릭합니다

API 키는 [Anthropic](https://console.anthropic.com/) 또는 [OpenAI](https://platform.openai.com/)에서 발급받을 수 있습니다. AI 비용은 사용량에 따라 제공업체에서 직접 청구됩니다 -- Loka는 추가 요금을 부과하지 않습니다.

---

### 휴대폰이나 태블릿에서 사용할 수 있나요?

네. Loka는 모바일 기기와 태블릿에서 작동하는 반응형 디자인의 웹 기반 플랫폼입니다. 별도의 모바일 앱은 없으며, 기기의 브라우저를 통해 Loka에 접근합니다.

모바일에서 최적의 경험을 위해 Chrome 또는 Safari 사용을 권장합니다.

---

### 데이터를 어떻게 내보내나요?

수업 요약은 수업 상세 페이지에서 **PDF** 파일로 개별 다운로드할 수 있습니다. 수업 내용의 오디오 버전은 **MP3** 파일로 생성하여 다운로드할 수 있습니다.

대량 데이터 내보내기(수업 이력, 학생 기록 등)의 경우, Loka Creator에게 문의하거나 고객지원에 연락하세요.

---

### 비밀번호를 잊었습니다. 어떻게 재설정하나요?

로그인 페이지에서 **비밀번호 찾기** 링크를 클릭합니다. 등록된 이메일 주소를 입력하면 비밀번호 재설정 링크가 발송됩니다. 링크는 제한된 시간 동안 유효합니다 -- 만료된 경우 새로 요청하면 됩니다.

이메일이 오지 않는 경우 스팸 폴더를 확인하세요. 문제가 지속되면 학교 관리자 또는 Loka 고객지원에 문의하세요.

---

### API 키와 데이터는 안전한가요?

네. Loka에 저장된 API 키는 저장 시 암호화됩니다. Loka는 모든 연결에 HTTPS를 사용하며, 민감한 데이터는 업계 표준 보안 관행에 따라 처리됩니다. AI API 키는 다른 사용자나 제3자와 공유되지 않으며, 오직 사용자를 대신하여 API 호출을 하는 데만 사용됩니다.
