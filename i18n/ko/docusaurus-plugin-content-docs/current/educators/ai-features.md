# AI 기반 기능

Loka는 AI를 연동하여 더 효과적으로 수업할 수 있도록 지원합니다. 교수 제안 생성부터 복습 자료 자동 생성까지, 이 도구들은 시간을 절약하고 학습 경험을 향상시킵니다.

## Bring Your Own Key (BYOK)

Loka는 AI 기능에 **BYOK 모델**을 사용합니다. 지원되는 AI 제공업체에서 직접 보유한 API 키를 제공하면, Loka가 해당 키를 사용하여 계정의 AI 기능을 실행합니다. 사용량에 따라 AI 제공업체에서 직접 청구되며, Loka는 AI 이용에 대해 추가 요금을 부과하거나 재판매하지 않습니다.

### 지원 제공업체

| 제공업체 | 키 설정 위치 | 기본 모델 |
|---------|------------|----------|
| **Anthropic Claude** | 설정 > AI 연동 | Claude Sonnet |
| **OpenAI** | 설정 > AI 연동 | GPT-4o Mini |

### API 키 추가

1. 교육자 대시보드에서 **설정**으로 이동합니다
2. **AI 연동** 섹션을 엽니다
3. Anthropic, OpenAI 또는 둘 다의 API 키를 입력합니다
4. **선호 제공업체**(Loka가 먼저 시도할 제공업체)를 선택합니다
5. 설정을 저장합니다

키는 암호화되어 안전하게 저장됩니다. Loka는 키를 제3자와 공유하지 않습니다.

:::tip
어떤 제공업체를 선택할지 모르겠다면, Anthropic Claude는 일반적으로 더 섬세한 언어 피드백을 생성하고, OpenAI GPT-4o Mini는 더 빠르고 토큰 사용이 적습니다. 둘 다 키를 추가하고 폴백 시스템이 알아서 처리하도록 할 수 있습니다.
:::

## 제공업체 폴백

두 제공업체의 키를 모두 보유한 경우, Loka는 자동 폴백 시스템을 사용합니다:

1. Loka가 먼저 **선호 제공업체**를 시도합니다
2. 해당 제공업체가 실패하면(속도 제한, 할당량 초과, 일시적 오류), Loka가 자동으로 **다른 제공업체**로 재시도합니다
3. 중단 없이 결과를 받을 수 있습니다

이는 단일 제공업체의 다운타임으로 인해 차단되는 일이 없음을 의미합니다.

## AI 기능

### AI 추천

수업 중 또는 수업 후 **AI 추천**을 사용하여 세션 노트를 기반으로 교수 권장 사항을 받을 수 있습니다:

- 수업 내용을 기반으로 한 후속 주제 제안
- 학생이 어려워한 영역에 대한 추천 연습
- 수업 주제와 관련된 어휘 확장 아이디어

Canvas 도구 모음 또는 세션 검토 화면에서 AI 추천에 접근할 수 있습니다.

### ARP 생성

AI가 수업 내용에서 자동으로 **Active Recall Pairs**를 생성할 수 있습니다:

1. [Living Canvas](/educators/canvas)에 노트가 있는 세션을 완료합니다
2. Canvas 메뉴에서 **ARP 생성**을 클릭합니다
3. AI가 세션 노트, 키워드, CIP를 분석합니다
4. 학생 복습용 질문-답변 쌍 세트가 생성됩니다

생성된 ARP를 학생의 복습 덱에 추가하기 전에 편집, 승인 또는 삭제할 수 있습니다.

### AI 기반 세션 피드백

수업 후 AI가 학생을 위한 구조화된 피드백을 생성할 수 있습니다:

- 수업 내용 요약
- 세션 중 보여준 강점
- 구체적인 예시와 함께 개선 영역
- 추천 연습 활동

이 피드백은 Canvas 노트와 CIP를 기반으로 하므로, 세션 기록이 상세할수록 피드백이 더 좋아집니다.

## Fathom AI 연동

회의 녹취를 위해 **Fathom AI**를 사용하는 경우 Loka에 연결할 수 있습니다:

1. **설정 > AI 연동**으로 이동합니다
2. Fathom API 키를 입력합니다
3. 화상 통화로 진행한 수업 후, Fathom의 녹취록이 Loka의 ARP 생성 및 피드백 도구에 입력될 수 있습니다

이 기능은 Canvas에 상세한 노트를 입력할 시간이 없는 회화 중심 수업에서 특히 강력합니다. 녹취록이 빈 부분을 채워줍니다.

## 사용량 및 비용

API 키를 직접 소유하므로, AI 비용은 사용량에 따라 달라집니다:

| 기능 | 일반적인 비용 |
|------|-------------|
| AI 추천 | 요청당 몇 센트 |
| ARP 생성 | 세션당 몇 센트 |
| 세션 피드백 | 세션당 몇 센트 |

비용은 AI 제공업체(Anthropic 또는 OpenAI)에서 직접 청구됩니다. 자세한 사용량 추적은 제공업체의 대시보드를 확인하세요.

:::note
AI 기능은 선택 사항입니다. Loka는 AI 없이도 완전히 작동합니다 -- ARP를 직접 만들고, 피드백을 작성하고, 수업을 수동으로 계획할 수 있습니다. AI는 단순히 그 과정을 빠르게 해줍니다.
:::
