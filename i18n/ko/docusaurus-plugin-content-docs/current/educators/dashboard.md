# 대시보드 개요

교육자 대시보드는 Loka의 총괄 관리 화면입니다. 여기서 수업을 시작하고, 학생을 관리하고, 수익을 추적하며, 모든 교수 도구에 접근할 수 있습니다.

## 대시보드 탭

대시보드는 세 가지 주요 탭으로 구성되어 있습니다:

### Canvas

**Canvas** 탭은 수업이 펼쳐지는 곳입니다. [Living Canvas](/educators/canvas)(Loka의 대화형 화이트보드)의 진입점으로, 여기서 다음을 수행할 수 있습니다:

- 학생과 새로운 세션 시작
- 지난 세션 Canvas 검토
- 저장된 노트, 키워드, Active Recall Pairs (ARP) 접근
- Canvas를 PDF 또는 MP3로 내보내기

### 세션

**세션** 탭은 예정된 수업과 지난 수업을 보여줍니다:

- **예정된 세션** -- 스케줄에 있는 수업 확인, 참여 또는 준비를 위한 빠른 링크 제공
- **지난 세션** -- 완료된 수업, 학생 출석, 세션 노트 검토
- **세션 이력** -- 날짜 범위, 학생, 서비스 유형별 필터링

### 관리

**관리** 탭은 비즈니스 측면을 관리합니다:

- 학생 명부 및 등록 상태
- 수익 요약 및 결제 이력
- 서비스 설정 (수업 유형, 가격, 시간)
- 학교 설정 및 브랜딩

## 주요 지표

대시보드 상단에서 선택한 기간의 한눈에 볼 수 있는 지표를 확인할 수 있습니다:

| 지표 | 설명 |
|------|------|
| **수익** | 완료된 수업의 총 매출 |
| **수업** | 진행한 세션 수 |
| **시간** | 총 교수 시간 |
| **학생** | 학교의 활성 학습자 수 |

날짜 범위 선택기를 사용하여 이번 주, 이번 달 또는 사용자 지정 범위의 지표를 확인할 수 있습니다.

## 빠른 액세스 기능

- **세션 시작** -- 대시보드에서 직접 새 수업을 시작합니다 (활성 [Loka Creator 구독](/educators/earnings) 필요)
- **캘린더 연동** -- 스케줄이 Google Calendar 및 iCal과 연동됩니다
- **알림** -- 예약 확인, 취소, 결제 알림이 알림 센터에 표시됩니다

## 시작하기

대시보드가 처음이라면:

1. 설정 탭에서 [프로필 설정](https://thelivingtextbook.lokalingo.com/teachers/profile)을 완료합니다
2. 학생이 수업을 예약할 수 있도록 [가능 시간을 설정](/educators/schedule)합니다
3. [AI 제공업체 키를 추가](/educators/ai-features)하여 AI 기반 교수 도구를 활성화합니다
4. 첫 학생을 초대하거나 예약 링크를 공유합니다

:::tip
대시보드 URL을 북마크하면 빠르게 접근할 수 있습니다. 휴대폰 홈 화면에 추가하면 네이티브 앱과 같은 경험을 얻을 수 있습니다.
:::
