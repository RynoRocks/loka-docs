# Living Canvas 사용법

**Living Canvas**는 Loka의 대화형 화이트보드로, 모든 수업의 핵심입니다. 세션 중 발생하는 모든 것(노트, 어휘, 교정, 학습 포인트)을 기록합니다. 매 수업마다 성장하는 살아있는 문서로 생각하세요.

## Living Canvas란?

수업 후 지워지는 정적 화이트보드와 달리, Living Canvas는 영구적으로 보존됩니다. 모든 노트, 키워드, 교정 내용이 자동으로 저장되며 학생의 학습 기록의 일부가 됩니다. 시간이 지나면 각 학습자에게 고유한 맞춤형 교재인 **Living Textbook**으로 구축됩니다.

## 세션 시작하기

1. [대시보드](/educators/dashboard)에서 **세션 시작**을 클릭합니다
2. 학생과 서비스 유형을 선택합니다
3. Living Canvas가 새 화면에서 열리며 수업 준비가 완료됩니다

:::note
세션을 시작하려면 활성 Loka Creator 구독이 필요합니다. 좌석 구매에 대한 메시지가 표시되면 [구독 상태](/educators/earnings)를 확인하세요.
:::

## Canvas 기능

### 그리기 및 입력

Canvas 도구 모음을 사용하여:

- **텍스트 입력** -- Canvas 어디든 클릭하여 입력을 시작합니다
- **자유 그리기** -- 다이어그램을 스케치하거나 연결을 강조하거나 주석을 추가합니다
- **섹션 추가** -- Canvas를 논리적 블록으로 구성합니다

### 키워드

중요한 어휘와 구문을 **키워드**로 기록합니다. 태그가 지정되어 검색이 가능하므로, 학생이 수업 후 특정 용어를 쉽게 검토할 수 있습니다.

### Active Recall Pairs (ARP)

ARP는 능동적 회상을 통해 학습을 강화하는 질문-답변 쌍입니다:

- 프롬프트(예: "'감사합니다'를 일본어로 어떻게 말하나요?")와 목표 답변을 입력하여 ARP를 추가합니다
- ARP는 학생 프로필에 저장되어 간격 반복 복습에 사용됩니다
- [AI 기능](/educators/ai-features)으로 수업 노트에서 자동으로 ARP를 생성할 수 있습니다

### 정확/부정확 구문 (CIP)

수업 중 표현을 **정확** 또는 **부정확**으로 표시합니다:

- **부정확한 구문** -- 학생이 실제로 말한 것
- **정확한 구문** -- 교정된 버전

CIP는 Living Textbook 방법론의 기초입니다. 대화의 실제 순간을 포착하여 맞춤형 학습 자료로 전환합니다.

### 자동 저장

Canvas는 작업 중 자동으로 저장됩니다. 노트를 잃어버릴 걱정이 없으며, 모든 것이 실시간으로 보존됩니다.

## Canvas 내보내기

세션 후 Canvas를 두 가지 형식으로 내보낼 수 있습니다:

### PDF 내보내기

세션 Canvas의 깔끔하고 인쇄 가능한 PDF를 생성합니다:

1. 완료된 세션의 Canvas를 엽니다
2. **내보내기** 메뉴를 클릭합니다
3. **PDF 다운로드**를 선택합니다

PDF에는 세션의 모든 텍스트, 키워드, ARP, CIP가 포함됩니다.

### MP3 내보내기 (텍스트 음성 변환)

AWS Polly를 사용하여 Canvas 텍스트를 오디오 파일로 변환합니다:

1. 완료된 세션의 Canvas를 엽니다
2. **내보내기** 메뉴를 클릭합니다
3. **MP3 생성**을 선택합니다
4. 대상 언어와 음성을 선택합니다
5. MP3가 생성되어 다운로드할 수 있습니다

이 기능은 발음 연습에 특히 유용합니다 -- 학생이 이동 중에 올바른 구문을 들을 수 있습니다.

:::tip
MP3 내보내기는 Canvas 텍스트가 깔끔하고 잘 구성되어 있을 때 가장 잘 작동합니다. 가장 명확한 오디오 출력을 위해 자유 그리기 노트보다 키워드와 CIP를 사용하세요.
:::

## Canvas 이력

모든 세션 Canvas는 저장되어 대시보드의 **Canvas** 탭에서 접근할 수 있습니다. 다음을 수행할 수 있습니다:

- 날짜 또는 학생별로 지난 세션 탐색
- 모든 Canvas를 다시 열어 검토하거나 노트를 계속 추가
- Canvas 이력을 검토하여 시간에 따른 학생 진도 추적

## 효과적인 Canvas 사용 팁

- **Canvas를 구조화하세요** -- 섹션을 사용하여 어휘, 문법, 대화 주제를 분리합니다
- **CIP를 실시간으로 기록하세요** -- 수업 후가 아니라 발생하는 즉시 교정 내용을 기록합니다
- **주요 학습 포인트에 ARP를 추가하세요** -- 각 세션에서 가장 중요한 3-5가지에 집중합니다
- **매 수업 후 내보내기** -- 학생에게 PDF 또는 MP3를 보내 독립적으로 복습할 수 있게 합니다
