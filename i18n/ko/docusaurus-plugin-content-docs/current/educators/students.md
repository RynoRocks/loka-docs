# 학생 관리

Loka는 학교 내 모든 학습자를 명확하게 볼 수 있게 해줍니다. 진도를 추적하고, 수업 이력을 검토하며, 등록을 한 곳에서 관리할 수 있습니다.

## 학생 목록

대시보드의 **관리** 탭에서 전체 학생 명부에 접근합니다. 학생 목록에는 다음이 표시됩니다:

- 학생 이름 및 프로필 사진
- 연락처 정보
- 등록 상태 (활성, 비활성, 대기)
- 완료한 총 수업 수
- 마지막 수업 날짜

검색창으로 이름이나 이메일로 학생을 검색하고, 필터로 상태나 활동 수준별로 범위를 좁힐 수 있습니다.

## 학생 프로필

학생을 클릭하면 전체 프로필을 볼 수 있습니다. 각 프로필에는 다음이 포함됩니다:

### 연락처 정보

- 이름, 이메일, 전화번호
- 선호 언어 및 시간대
- 학습 목표에 대해 추가한 노트

### 학습 이력

학습 과정의 전체 기록:

- **세션 로그** -- 날짜, 시간, 서비스 유형이 포함된 모든 수업
- **Canvas 이력** -- 노트, 키워드, CIP가 포함된 모든 [Living Canvas](/educators/canvas) 세션
- **ARP** -- 지난 세션의 전체 [Active Recall Pairs](/educators/canvas#active-recall-pairs-arp) 라이브러리
- **진도 추이** -- 시간에 따른 성과 변화

### 예약 이력

모든 과거 및 예정 예약 확인:

- 수업 날짜 및 시간
- 서비스 유형 및 시간
- 결제 상태
- 취소 이력

## 학습자가 학교에 참여하는 방법

학생이 등록할 수 있는 여러 가지 방법이 있습니다:

### 예약 링크

학교의 예약 페이지 URL을 예비 학생에게 공유합니다. 다음을 수행할 수 있습니다:

1. 예약 페이지를 방문합니다
2. 이용 가능한 서비스와 시간 슬롯을 탐색합니다
3. Loka 계정을 생성합니다 (또는 로그인)
4. 첫 수업을 예약하고 결제를 완료합니다

예약 후 학생은 자동으로 학교 명부에 추가됩니다.

### 직접 초대

대시보드에서 학생을 직접 초대합니다:

1. **관리 > 학생**으로 이동합니다
2. **학생 초대**를 클릭합니다
3. 이름과 이메일 주소를 입력합니다
4. 계정 설정 링크가 포함된 초대 이메일이 발송됩니다

### 관리자 배정

한 학교에서 여러 교육자를 관리하는 경우, 관리자가 관리 패널을 통해 학생을 특정 교사에게 배정할 수 있습니다.

## 학습자 좌석

각 활성 학생에게는 **학습자 좌석** 구독이 필요합니다. 주요 내용:

- 학습자 좌석은 학생당 $6/월입니다
- 학습자가 수업료를 결제하면 좌석이 자동으로 갱신됩니다
- 학생이 수업 예약을 중단하면 좌석 구독도 중단됩니다
- 활성 좌석이 없는 학생은 새 수업을 예약할 수 없습니다

:::note
학습자 좌석은 수업 결제에 연동됩니다 -- 수동으로 관리할 필요가 없습니다. 학생이 예약하고 결제하면 좌석이 활성 상태를 유지합니다. 중단하면 자연스럽게 만료됩니다.
:::

## 학생 기록 관리

### 노트 추가

학생 프로필에 비공개 노트를 추가하여 다음을 기록합니다:

- 학습 목표 및 목적
- 선호하는 교수 스타일 또는 속도
- 향후 세션에서 다시 다룰 주제

이 노트는 교육자만 볼 수 있으며 학생에게는 보이지 않습니다.

### 진도 검토

학생의 Canvas 이력과 ARP 라이브러리를 사용하여 진도를 평가합니다:

- CIP 추이를 확인합니다 -- 시간이 지남에 따라 실수가 줄어들고 있나요?
- ARP 숙달도를 검토합니다 -- 어떤 회상 쌍을 내면화했나요?
- 세션 빈도를 확인합니다 -- 정기적으로 예약하고 있나요?

### 학생 비활성화

학생이 더 이상 활성 상태가 아닌 경우:

- 프로필은 기록 보관용으로 명부에 남아 있습니다
- 과거 세션 데이터, Canvas, ARP가 보존됩니다
- 새 수업을 예약하면 언제든 다시 활성화할 수 있습니다
