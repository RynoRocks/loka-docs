# 무료 체험 계정 만들기

Loka 시작은 2분도 걸리지 않습니다. 무료 체험으로 플랫폼의 모든 기능을 이용할 수 있으므로, 플랜을 결정하기 전에 모든 것을 둘러볼 수 있습니다.

## 가입 방법

1. [thelivingtextbook.lokalingo.com/start-free](https://thelivingtextbook.lokalingo.com/start-free)에 접속합니다
2. 이름, 이메일 주소, 비밀번호를 입력합니다
3. 인증 링크를 통해 이메일 주소를 확인합니다
4. 로그인하면 교육자 대시보드로 이동합니다

간단합니다. 신용카드도, 상담 전화도, 설정 비용도 필요 없습니다.

## 포함 사항

14일 무료 체험에는 다음이 포함됩니다:

- **교육자 좌석 1개** -- 교육자 대시보드, 수업 도구, 스케줄링, Living Canvas에 대한 전체 액세스
- **학습자 테스트 좌석 1개** -- 양쪽 입장에서 플랫폼을 경험할 수 있는 테스트 학습자 계정
- **모든 핵심 기능:**
  - 세션 생성 및 관리
  - 수업 중 Living Canvas 사용
  - 가능 시간 설정 및 예약 페이지 구성
  - Living Textbook 이용 (자동 생성 학습자 노트)
  - AI 기능 설정 (직접 보유한 API 키 사용)
  - 분석 및 수업 이력 확인

:::info 체험 기간 중 제한 없음
체험은 제한된 데모가 아닙니다. 유료 DIY 플랜과 동일한 기능을 이용할 수 있습니다. 유일한 제한은 기간(14일)뿐입니다.
:::

## 체험 종료 후

14일 무료 체험이 만료되면:

- **데이터는 보존됩니다.** 세션, 학습자 기록, Canvas 이력, 설정 모두 그대로 유지됩니다.
- **활성 기능이 일시 중지됩니다.** 업그레이드하기 전까지 새로운 세션을 시작하거나 Canvas에 접근할 수 없습니다.
- **언제든 업그레이드할 수 있습니다.** 플랜을 선택하면 모든 데이터가 그대로 유지된 채 계정이 즉시 활성화됩니다.

:::warning 흐름을 놓치지 마세요
14일째 전에 플랜을 결정하여 수업이 중단되지 않도록 하는 것을 권장합니다. 각 옵션에 대한 자세한 내용은 [플랜 업그레이드](./upgrading)를 참고하세요.
:::

## 테스트 학습자 초대하기

체험 기간 중 실제 학습자를 초대하거나 포함된 테스트 좌석을 사용하여 학습자 경험을 시뮬레이션할 수 있습니다:

1. 교육자 대시보드에서 **학습자 > 학습자 초대**로 이동합니다
2. 학습자의 이메일 주소를 입력합니다 (테스트용으로 본인의 보조 이메일을 사용해도 됩니다)
3. 학습자에게 계정 생성 링크가 포함된 초대 이메일이 발송됩니다
4. 초대를 수락하면 학습자 목록에 표시되고 세션을 예약할 수 있습니다

이 방법이 실제 학생과 함께하기 전에 전체 예약 및 수업 흐름을 확인하는 가장 좋은 방법입니다.
