# 플랜 업그레이드

무료 체험이 종료되면(또는 준비가 되면) 업그레이드는 약 30초면 완료됩니다. 교수 방식에 맞는 플랜을 선택하면 계정이 즉시 활성화됩니다.

## 플랜 안내

Loka는 교육자를 위한 세 가지 플랜을 제공하며, 모두 Stripe를 통해 안전하게 결제됩니다.

### Loka Creator DIY -- $6/월

직접 설정을 관리하는 독립 교육자를 위한 필수 플랜입니다.

**포함 사항:**
- 교육자 좌석 1개
- 학습자 테스트 좌석 1개
- Living Canvas 전체 이용
- Living Textbook (자동 생성 학습자 노트)
- 스케줄링 및 예약 시스템
- Stripe 결제 연동
- AI 기능 (직접 보유한 API 키 사용)
- 분석 및 수업 이력
- Canvas 내보내기 (PDF, MP3)

**적합 대상:** 가볍고 합리적인 가격의 플랫폼을 원하며 직접 설정하는 것을 선호하는 독립 교육자.

---

### Loka Creator DFY -- $29/월

DIY의 모든 기능에 설정 지원이 추가됩니다.

**DIY의 모든 기능 포함, 추가 사항:**
- Done-For-You 초기 설정 -- 계정, 스케줄, 예약 페이지를 대신 설정해 드립니다
- 월 2시간 기술 지원
- 우선 이메일 지원

**적합 대상:** 기술 설정보다 수업에 집중하고 싶은 교육자. 다른 플랫폼에서 전환하거나 새로 시작하며 가이드가 필요한 경우에 적합합니다.

---

### Loka Creator Aspire -- $199/월

교육 비즈니스를 구축하는 교육자를 위한 풀 패키지입니다.

**DFY의 모든 기능 포함, 추가 사항:**
- 월 2시간 파운더 직접 코칭
- 비즈니스 전략 세션 (가격 책정, 포지셔닝, 성장)
- 맞춤 설정 및 워크플로우 최적화
- 신규 기능 조기 액세스

**적합 대상:** 1:1 수업을 넘어 확장하고자 하는 야심 있는 교육자 -- 팀 구축, 브랜드 구축 또는 코스 기반 비즈니스 런칭.

## 추가 좌석

교육 사업이 성장함에 따라 좌석을 추가할 수 있습니다:

- **교육자 좌석** -- 좌석당 $6/월. 자체 대시보드, 스케줄, Canvas가 필요한 팀원을 추가합니다.
- **학습자 좌석** -- 좌석당 $6/월. 학습자가 수업료를 결제하면 자동으로 갱신됩니다. 학습자가 예약을 중단하면 좌석 구독도 자동으로 중지됩니다.

:::info 학습자 좌석은 활동에 연동됩니다
교육자 좌석(표준 월간 구독)과 달리 학습자 좌석은 학습자가 수업료를 결제할 때만 갱신됩니다. 이는 비활성 학습자에 대해 비용을 지불하지 않음을 의미합니다.
:::

## 업그레이드 방법

업그레이드하는 두 가지 방법이 있습니다:

### 체험 배너에서

체험 기간 중 대시보드 상단에 잔여 일수를 보여주는 배너가 표시됩니다. 배너의 **업그레이드** 버튼을 클릭하면 플랜 선택 페이지로 바로 이동합니다.

### 설정에서

1. 교육자 대시보드에 로그인합니다
2. **설정 > 구독**으로 이동합니다
3. 플랜을 선택합니다
4. 결제 정보를 입력합니다 (Stripe를 통해 안전하게 처리)
5. 계정이 즉시 업그레이드됩니다

## 데이터는 어떻게 되나요?

**모든 데이터가 보존됩니다.** 체험에서 업그레이드하거나 플랜을 변경할 때:

- 모든 세션과 Canvas 이력이 그대로 유지됩니다
- 학습자 기록과 진도가 보존됩니다
- 스케줄 및 가능 시간 설정이 이어집니다
- AI 설정 및 API 키가 변경되지 않습니다
- 분석 및 리포트가 중단 없이 계속됩니다

:::tip 플랜 전환
언제든 플랜을 업그레이드하거나 다운그레이드할 수 있습니다. 변경 사항은 다음 결제 주기 시작 시 적용됩니다. 플랜 전환 시 데이터는 절대 삭제되지 않습니다.
:::

## 자주 묻는 질문

**결제 전에 체험해 볼 수 있나요?**
네. [14일 무료 체험](./free-trial)으로 신용카드 없이 모든 DIY 기능을 이용할 수 있습니다.

**계약이나 약정이 있나요?**
없습니다. 모든 플랜은 월 단위 결제입니다. 설정 페이지에서 언제든 취소할 수 있습니다.

**어떤 결제 수단을 사용할 수 있나요?**
Stripe를 통해 모든 주요 신용카드 및 직불카드를 지원합니다. Aspire 플랜 고객은 요청 시 청구서 결제도 가능합니다.

**환불이 가능한가요?**
유료 플랜 가입 후 30일 이내에 만족하지 못하신다면 [문의하기](https://lokalingo.com/en/contact)로 연락 주시면 해결해 드리겠습니다.

**AI 기능에 별도의 API 키가 필요한가요?**
네. Loka는 AI에 Bring Your Own Key (BYOK) 모델을 사용합니다. Anthropic Claude 또는 OpenAI API 키를 제공하고 제공업체에 직접 비용을 지불합니다. 이를 통해 AI 비용이 투명하고 직접 통제할 수 있습니다.
