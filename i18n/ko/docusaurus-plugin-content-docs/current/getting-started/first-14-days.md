# 처음 14일

이 가이드는 무료 체험 기간을 최대한 활용하는 방법을 안내합니다. 14일이 끝나면 완전히 구성된 교수 환경을 갖추고, Loka가 본인의 교육 방식에 적합한지 명확하게 판단할 수 있을 것입니다.

## 1일차: 프로필 설정

먼저 Loka를 본인에게 맞게 설정하세요.

- **교육자 프로필 완성** -- 이름, 사진, 소개, 교수 전문 분야를 추가합니다. 이 정보는 예약 페이지에 표시됩니다.
- **대시보드 둘러보기** -- 10분 정도 주요 섹션(세션, 학습자, 스케줄, Canvas, 설정)을 클릭하며 살펴보세요.
- **시간대 설정** -- Loka는 모든 스케줄링에 설정된 시간대를 사용합니다. 설정에서 확인하세요.

:::tip 첫인상이 중요합니다
프로필은 학습자가 예약 페이지에 방문했을 때 보게 되는 정보입니다. 전문적인 사진과 간결한 소개글이 큰 차이를 만듭니다.
:::

## 2-3일차: 스케줄 설정

학습자가 예약할 수 있도록 가능 시간을 설정합니다.

- **주간 가능 시간 설정** -- **스케줄 > 가능 시간**으로 이동하여 정기 오픈 슬롯을 정의합니다(예: 월-금, 오전 9시-오후 5시).
- **예외 시간 설정** -- 휴일, 회의, 개인 시간 등 특정 날짜나 시간을 비가용으로 표시합니다.
- **캘린더 연동** -- Google Calendar과 연동하여 이중 예약을 방지합니다. **설정 > 연동 > Google Calendar**로 이동합니다.
- **데모 세션 만들기** -- 포함된 학습자 테스트 좌석으로 테스트 세션을 예약하여 양쪽에서 예약 흐름이 어떻게 작동하는지 확인합니다.

## 4-5일차: 첫 세션 진행

Loka의 핵심인 실시간 수업 세션을 체험합니다.

- 대시보드에서 예약된 수업 옆의 **세션 시작** 버튼을 클릭하여 **세션을 시작**합니다.
- **Living Canvas를 사용해 보세요** -- 대화형 화이트보드입니다. 세션 중 다음을 할 수 있습니다:
  - 교정 사항과 어휘 입력
  - 정확/부정확 표시
  - 노트와 예문 추가
  - 주제별 내용 정리
- **세션을 종료**하고, 학습자에게 전달되는 자동 생성 Living Textbook 항목을 확인합니다.

:::info 혼자서 먼저 연습하세요
Canvas를 테스트하기 위해 실제 학습자가 필요하지 않습니다. 테스트 좌석으로 세션을 시작하고 도구를 실험해 보세요. 이렇게 하면 실제 학생과 함께할 때 자신감을 가질 수 있습니다.
:::

## 6-7일차: 첫 실제 학습자 초대

실제 학생을 초대할 차례입니다.

- **초대 발송** -- **학습자 > 학습자 초대**로 이동하여 이메일 주소를 입력합니다.
- **예약 링크 공유** -- 각 교육자에게 고유 예약 URL이 있습니다. 학습자에게 보내면 시간대를 선택할 수 있습니다.
- **실제 세션 진행** -- Living Canvas를 사용하여 전체 수업을 진행합니다. 교정 내용이 어떻게 기록되고 세션 요약이 어떻게 생성되는지 주목하세요.
- **피드백 요청** -- 세션 후 학습자에게 예약 및 수업 경험에 대한 의견을 물어보세요. 이른 시기의 피드백은 매우 유용합니다.

## 2주차: 고급 기능 탐색

기본 사항을 익혔다면, Loka만의 차별화된 기능을 살펴보세요.

### AI 기능 설정

Loka는 직접 보유한 API 키를 사용하여 AI 기반 수업 준비 및 피드백을 지원합니다.

1. **설정 > AI 연동**으로 이동합니다
2. **Anthropic Claude** 또는 **OpenAI**(또는 둘 다) API 키를 입력합니다
3. 선호하는 제공업체를 선택합니다
4. Canvas에서 **AI 추천**을 시도합니다 -- 세션 중 맥락에 맞는 교수 제안을 생성합니다
5. 수업 종료 후 AI 생성 세션 피드백을 확인합니다

:::info Bring Your Own Key
Loka는 AI 기능에 대해 별도의 요금을 부과하지 않습니다. 직접 보유한 API 키를 사용하고 제공업체에 직접 비용을 지불합니다. 이렇게 하면 비용이 투명하고 통제 가능합니다.
:::

### 분석 확인

- 대시보드의 **분석**에서 세션 수, 학습자 활동, 교수 패턴을 확인합니다
- 개별 학습자 진도와 Living Textbook 이력을 검토합니다
- 가장 바쁜 시간대를 확인하여 스케줄을 최적화합니다

### Canvas 기능 탐색

- **PDF 내보내기** -- 오프라인 검토를 위해 Canvas 세션의 PDF를 생성합니다
- **MP3 내보내기** -- 텍스트 음성 변환(AWS Polly 기반)으로 Canvas 텍스트를 오디오로 변환합니다
- **음성 녹음** -- 세션 중 Canvas에서 직접 오디오를 녹음합니다

## 13-14일차: 플랜 결정

체험이 만료되기 전에 지금까지 구축한 것을 점검하세요.

**스스로에게 물어보세요:**
- 수업 흐름이 자연스러웠는가?
- 학습자들이 예약 과정을 쉽게 느꼈는가?
- 지속적으로 AI 기능을 사용하고 싶은가?
- 학습자 좌석이 몇 개 필요한가?

**선택 가능한 플랜:**

| 플랜 | 가격 | 적합 대상 |
|------|------|----------|
| **DIY** | $6/월 | 모든 것을 직접 관리하는 독립 교육자 |
| **DFY** | $29/월 | 설정 지원과 기술 지원을 원하는 교육자 |
| **Aspire** | $199/월 | 파운더 코칭과 함께 교육 비즈니스를 구축하는 교육자 |

각 플랜의 상세 내용은 [플랜 업그레이드](./upgrading)를 참고하세요.

:::tip 데이터는 안전합니다
체험 기간 중 생성한 모든 것(세션, 학습자 기록, Canvas 이력, 설정)은 업그레이드 시 그대로 유지됩니다. 아무것도 사라지지 않습니다.
:::
