# 계정 설정

대시보드의 **계정 설정** 페이지에서 프로필, 환경 설정, 보안 설정을 관리할 수 있습니다.

## 프로필 관리

프로필에는 교사와 학교에 보이는 기본 정보가 포함됩니다. 다음을 업데이트할 수 있습니다:

- **표시 이름** -- 교사와 수업 기록에 표시되는 이름
- **프로필 사진** -- 사진을 업로드하여 계정을 개인화합니다
- **이메일 주소** -- 로그인 및 알림에 사용되는 주소. 이메일 변경 시 재인증이 필요합니다
- **전화번호** -- 선택 사항. 학교에서 SMS 알림을 활성화한 경우 사용됩니다
- **소개 / 자기소개** -- 교사가 볼 수 있는 짧은 소개. 관심사와 목표에 맞게 수업을 조정하는 데 도움이 됩니다

프로필을 업데이트하려면:

1. 오른쪽 상단의 이름이나 아바타를 클릭합니다
2. **프로필** 또는 **계정 설정**을 선택합니다
3. 변경 사항을 입력하고 **저장**을 클릭합니다

## 시간대 설정

Loka는 모든 수업 시간을 현지 시간대로 표시합니다. 올바르게 설정하는 것이 예약 혼동을 방지하는 데 중요합니다.

- 처음 가입 시 시간대가 자동으로 감지됩니다
- **계정 설정 > 시간대**에서 수동으로 변경할 수 있습니다
- 모든 예약 확인, 알림, 캘린더 항목은 선택한 시간대를 사용합니다

:::caution
수업 시간이 잘못 표시되는 경우, 가장 흔한 원인은 잘못된 시간대 설정입니다. 고객지원에 문의하기 전에 이 설정을 먼저 확인하세요.
:::

## 비밀번호 변경

비밀번호를 변경하려면:

1. **계정 설정 > 보안**으로 이동합니다
2. 현재 비밀번호를 입력합니다
3. 새 비밀번호를 입력하고 확인합니다
4. **비밀번호 변경**을 클릭합니다

**비밀번호 요구 사항:**
- 최소 8자
- 대문자, 소문자, 숫자 각각 최소 1개 포함

비밀번호를 잊은 경우 로그인 페이지의 **비밀번호 찾기** 링크를 사용하세요. 등록된 이메일 주소로 재설정 링크가 발송됩니다.

## 알림 설정

Loka는 수업, 자료, 계정 활동에 대해 알림을 보냅니다. 수신할 알림을 제어할 수 있습니다:

### 이메일 알림

- **예약 확인** -- 수업을 예약하거나 취소할 때 발송
- **수업 알림** -- 예정된 수업 전에 발송 (예: 24시간 전, 1시간 전)
- **새 자료 알림** -- 교사가 수업 요약을 공개할 때 발송
- **과제 알림** -- 미완료 숙제가 있을 때 발송
- **계정 업데이트** -- 비밀번호 변경, 이메일 변경, 보안 알림

### 설정 관리

1. **계정 설정 > 알림**으로 이동합니다
2. 각 알림 유형을 켜거나 끕니다
3. 변경 사항은 즉시 적용됩니다

:::tip
**수업 알림**과 **새 자료 알림**은 켜두는 것을 권장합니다. 적시에 받는 알림은 수업 준비에 도움이 되며, 수업 직후 자료를 검토하면 기억력이 크게 향상됩니다.
:::
