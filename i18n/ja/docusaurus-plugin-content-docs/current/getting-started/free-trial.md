# 無料トライアルアカウントの作成

Lokaを始めるのに必要な時間は2分もかかりません。無料トライアルでは、プラットフォームのすべての機能にアクセスでき、プランを選ぶ前にじっくり試すことができます。

## サインアップ方法

1. [thelivingtextbook.lokalingo.com/start-free](https://thelivingtextbook.lokalingo.com/start-free) にアクセス
2. 名前、メールアドレス、パスワードを入力
3. 確認メールに記載されたリンクでメールアドレスを認証
4. ログインして、エデュケーターダッシュボードに到着

以上で完了です。クレジットカード不要、営業電話なし、初期費用なしで始められます。

## トライアルに含まれる内容

14日間の無料トライアルには以下が含まれます。

- **エデュケーターシート1席** -- エデュケーターダッシュボード、セッションツール、スケジュール管理、Living Canvasへのフルアクセス
- **ラーナーテストシート1席** -- テスト用のラーナーアカウントで、プラットフォームを両方の視点から体験可能
- **すべてのコア機能:**
  - セッションの作成・管理
  - レッスン中のLiving Canvas使用
  - 空き状況と予約ページの設定
  - Living Textbook（自動生成ラーナーノート）へのアクセス
  - AI機能の設定（自前のAPIキーを使用）
  - 分析とセッション履歴の閲覧

:::info トライアル中の制限はありません
トライアルは機能制限付きのデモではありません。有料DIYプランと同じ機能が利用できます。唯一の制限は期間のみ -- 14日間です。
:::

## トライアル終了後はどうなるか

14日間のトライアル期間が終了すると、以下のようになります。

- **データは保持されます。** セッション、ラーナー記録、Canvas履歴、設定はすべてそのまま残ります。
- **アクティブな機能は一時停止されます。** アップグレードするまで、新しいセッションの開始やCanvasへのアクセスはできなくなります。
- **いつでもアップグレード可能です。** プランを選択すれば、すべてのデータがそのままの状態ですぐにアカウントが再有効化されます。

:::warning 勢いを止めないために
トライアル14日目までにプランを決定しておくことをお勧めします。指導に空白期間が生じないようにするためです。各プランの詳細は[プランのアップグレード](./upgrading)をご確認ください。
:::

## テストラーナーの招待

トライアル期間中は、実際のラーナーを招待するか、付属のテストシートを使ってラーナー体験をシミュレートできます。

1. エデュケーターダッシュボードから **ラーナー > ラーナーを招待** に移動
2. ラーナーのメールアドレスを入力（テスト用に自分の別メールアドレスも使用可能）
3. ラーナーにアカウント作成リンク付きの招待メールが届きます
4. 招待を承認すると、ラーナーリストに表示され、セッションの予約が可能になります

本番のラーナーで運用を始める前に、予約からセッションまでの全体の流れを確認する最良の方法です。
