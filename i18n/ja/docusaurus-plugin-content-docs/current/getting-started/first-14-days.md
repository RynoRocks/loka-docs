# はじめの14日間

このガイドでは、無料トライアルを最大限に活用する方法を説明します。14日間が終わる頃には、指導環境が完全に整い、Lokaが自分の教育スタイルに合っているかどうか明確に判断できるようになります。

## 1日目：プロフィールを設定する

まずはLokaを自分仕様にカスタマイズしましょう。

- **エデュケータープロフィールを完成させる** -- 名前、写真、自己紹介、専門分野を追加します。この情報は予約ページに表示されます。
- **ダッシュボードを探索する** -- セッション、ラーナー、スケジュール、Canvas、設定の各セクションを10分程度かけてクリックしてみてください。
- **タイムゾーンを設定する** -- Lokaはすべてのスケジュール管理にタイムゾーンを使用します。設定画面で正しく設定されているか確認してください。

:::tip 第一印象が大切
プロフィールは、ラーナーが予約ページに訪れた際に最初に目にする情報です。プロフェッショナルな写真と簡潔な自己紹介があるだけで、印象は大きく変わります。
:::

## 2〜3日目：スケジュールを設定する

ラーナーが予約できるように、空き状況を設定します。

- **週間スケジュールを設定する** -- **スケジュール > 空き状況** に移動し、定期的な空き時間を定義します（例：月曜〜金曜、9時〜17時）。
- **例外日を設定する** -- 祝日、会議、プライベートの時間など、特定の日付や時間帯を「対応不可」に設定します。
- **カレンダーを連携する** -- Google Calendarと同期して、ダブルブッキングを防ぎます。**設定 > 連携 > Google Calendar** から設定できます。
- **デモセッションを作成する** -- 付属のラーナーテストシートを使って、予約の流れを両方の視点から確認するテストセッションを予約してみましょう。

## 4〜5日目：最初のセッションを実施する

Lokaの中核機能 -- ライブティーチングセッションを体験しましょう。

- ダッシュボードから、予約済みのレッスン横にある **セッション開始** をクリックしてセッションを開始します。
- **Living Canvasを試す** -- インタラクティブなホワイトボードです。セッション中に以下のことができます：
  - 添削内容や語彙を入力
  - 正解・不正解の項目をマーク
  - メモや例文を追加
  - トピックごとにコンテンツを整理
- **セッションを終了** し、ラーナーに届く自動生成のLiving Textbookエントリを確認します。

:::info まずはソロで練習を
Canvasのテストに実際のラーナーは必要ありません。テストシートでセッションを開始し、ツールを試してみてください。本番で自信を持って使えるようになります。
:::

## 6〜7日目：最初の実ラーナーを招待する

実際の生徒を招待してみましょう。

- **招待を送信する** -- **ラーナー > ラーナーを招待** で、メールアドレスを入力します。
- **予約リンクを共有する** -- 各エデュケーターに固有の予約URLがあります。ラーナーに送信して、希望の時間帯を選んでもらいましょう。
- **実際のセッションを実施する** -- Living Canvasを使って完全なレッスンを行います。添削がどのようにキャプチャされ、セッションサマリーがどのように生成されるかに注目してください。
- **フィードバックを求める** -- セッション後、ラーナーに予約とセッションの体験について感想を聞きましょう。早期のフィードバックはとても貴重です。

## 2週目：高度な機能を探索する

基本を押さえたら、Lokaならではの機能をさらに掘り下げましょう。

### AI機能を設定する

LokaはAIによるレッスン準備やフィードバックを、自前のAPIキーでサポートしています。

1. **設定 > AI連携** に移動
2. **Anthropic Claude** または **OpenAI**（もしくは両方）のAPIキーを入力
3. 優先プロバイダーを選択
4. Canvasで **AI Suggest** を試す -- セッション中に文脈に応じた指導提案を生成します
5. レッスン終了後にAI生成のセッションフィードバックを確認

:::info 自前のキーを使用（BYOK）
LokaではAI機能に料金はかかりません。自前のAPIキーを使い、プロバイダーに直接支払います。AIコストが透明で、自分でコントロールできます。
:::

### 分析データを確認する

- ダッシュボードの **分析** で、セッション数、ラーナーの活動状況、指導パターンを確認
- 個々のラーナーの進捗やLiving Textbookの履歴をレビュー
- 最も予約が集中する時間帯を確認し、スケジュールを最適化

### Canvas機能を探索する

- **PDFエクスポート** -- Canvasセッションのオフラインレビュー用にPDFを生成
- **MP3エクスポート** -- テキスト読み上げ機能（AWS Polly）でCanvasテキストを音声に変換
- **音声録音** -- セッション中にCanvas内で直接音声を録音

## 13〜14日目：プランを決定する

トライアル期限前に、これまでの成果を振り返りましょう。

**確認ポイント：**
- セッションの流れは自然に感じたか？
- ラーナーにとって予約プロセスは簡単だったか？
- AI機能を継続的に使いたいか？
- ラーナーシートは何席必要か？

**選べるプラン：**

| プラン | 料金 | 最適な対象 |
|------|-------|----------|
| **DIY** | $6/月 | すべてを自分で管理する独立エデュケーター |
| **DFY** | $29/月 | セットアップのサポートとテクニカルサポートを求めるエデュケーター |
| **Aspire** | $199/月 | 創業者コーチング付きで教育ビジネスを構築するエデュケーター |

各プランの詳細は[プランのアップグレード](./upgrading)をご覧ください。

:::tip データは安全に保持されます
トライアル中に作成したすべてのデータ -- セッション、ラーナー記録、Canvas履歴、設定 -- は、アップグレード時にそのまま引き継がれます。何も失われません。
:::

## もっと時間が必要な場合

14日間では足りなかった場合は、[お問い合わせ](https://lokalingo.com/en/contact)ください。急いだ判断よりも、納得のいく判断をしていただきたいと考えています。
