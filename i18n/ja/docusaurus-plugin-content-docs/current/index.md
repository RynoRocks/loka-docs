# Lokaへようこそ

Lokaは、自分のスタイルで語学教育ビジネスを運営したいエデュケーター向けの学習管理システムです。マンツーマンの会話レッスン、グループクラス、その他あらゆる形式の授業に対応し、生徒管理、セッション進行、決済までをひとつのプラットフォームで完結できます。

## Lokaでできること

- **Living Canvas** -- リアルタイムで操作できるインタラクティブなホワイトボードで、すべてのレッスンを「生きたドキュメント」に変えます。指導中に添削、語彙、メモを記録し、ラーナーと即座に共有できます。
- **Living Textbook** -- 添削内容やティーチングモーメントが自動的に蓄積され、ラーナーごとにパーソナライズされたテキストブックが作られます。メモの紛失やフィードバックの忘れはもうありません。
- **スケジュール＆予約** -- ラーナーが空き状況から直接予約できます。カレンダー同期機能で、すべてのスケジュールを整理できます。
- **決済＆サブスクリプション** -- Stripeを活用した課金システムが、レッスン料、シートサブスクリプション、クレジットパックの処理を自動で行います。
- **AI連携** -- 自前のAPIキー（Anthropic ClaudeまたはOpenAI）を使って、AIによるレッスン準備、リアルタイム提案、セッション後のフィードバックを利用できます。
- **分析＆進捗管理** -- ラーナーの進捗、セッション履歴、指導パターンを時系列で追跡できます。

## Lokaの対象ユーザー

Lokaは3つのタイプのユーザーを対象としており、それぞれ専用のドキュメントがあります。

| ユーザータイプ | 説明 | ガイド |
|-----------|-------------|-------|
| **エデュケーター** | レッスンを実施し、ラーナーを管理する語学教師 | [エデュケーターガイド](./educators/dashboard) |
| **ラーナー** | レッスンを予約し、セッションに参加し、進捗を確認する学習者 | [ラーナーガイド](./learners/booking) |
| **Loka Creator** | 自分のブランドで語学教育ビジネスを構築・拡大したいエデュケーター | [クリエイターガイド](./creators/overview) |
