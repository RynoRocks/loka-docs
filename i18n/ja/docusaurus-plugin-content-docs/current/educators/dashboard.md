# ダッシュボードの概要

エデュケーターダッシュボードは、Lokaにおける指揮センターです。レッスンの開始、生徒の管理、収益の追跡、すべての指導ツールへのアクセスがここから行えます。

## ダッシュボードのタブ

ダッシュボードは3つのメインタブで構成されています。

### Canvas

**Canvas** タブは、レッスンが形になる場所です。ここからLokaのインタラクティブなホワイトボードである[Living Canvas](/educators/canvas)にアクセスできます。以下のことが可能です：

- 生徒との新しいセッションを開始
- 過去のセッションCanvasを確認
- 保存済みのメモ、キーワード、Active Recall Pairs（ARP）にアクセス
- CanvasをPDFまたはMP3でエクスポート

### セッション

**セッション** タブには、今後のレッスンと過去のレッスンが表示されます：

- **今後のセッション** -- スケジュール上の予定を確認し、参加や準備へのクイックリンクを利用
- **過去のセッション** -- 完了したレッスン、生徒の出席状況、セッションメモを確認
- **セッション履歴** -- 日付範囲、生徒、サービスタイプでフィルタリング

### マネジメント

**マネジメント** タブでは、ビジネス面の管理ができます：

- 生徒名簿と在籍状況
- 収益サマリーと支払い履歴
- サービス設定（レッスンタイプ、価格、所要時間）
- スクール設定とブランディング

## 主要指標

ダッシュボード上部には、選択期間の一目でわかる指標が表示されます：

| 指標 | 説明 |
|--------|-------------|
| **収益** | 完了したレッスンからの合計売上 |
| **レッスン数** | 実施したセッション数 |
| **所要時間** | ログされた指導時間の合計 |
| **生徒数** | スクール内のアクティブなラーナー数 |

日付範囲ピッカーを使って、今週、今月、またはカスタム範囲の指標を確認できます。

## クイックアクセス機能

- **セッション開始** -- ダッシュボードから直接新しいレッスンを開始（アクティブな[Loka Creatorサブスクリプション](/educators/earnings)が必要）
- **カレンダー同期** -- Google CalendarおよびiCalとスケジュールを連携
- **通知** -- 予約確認、キャンセル、支払いアラートが通知センターに表示されます

## はじめに

ダッシュボードを初めて使う場合：

1. 設定タブで[プロフィール設定](https://thelivingtextbook.lokalingo.com/teachers/profile)を完了する
2. [空き状況を設定](/educators/schedule)して、生徒がレッスンを予約できるようにする
3. [AIプロバイダーキーを追加](/educators/ai-features)して、AI搭載の指導ツールを有効化する
4. 最初の生徒を招待するか、予約リンクを共有する

:::tip
ダッシュボードのURLをブックマークしておくと、すばやくアクセスできます。スマートフォンのホーム画面に追加すれば、ネイティブアプリのような体験も可能です。
:::
