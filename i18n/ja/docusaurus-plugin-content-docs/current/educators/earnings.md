# 収益と支払い

Lokaは指導による収益を追跡し、収益状況を簡単に把握できるようにします。サマリーの確認、詳細の掘り下げ、支払い方法の設定が可能です。

## 収益の確認

ダッシュボードの **マネジメント** タブに移動して、収益の概要にアクセスします。以下の情報が確認できます：

### 収益サマリー

選択期間の収益のハイレベルビュー：

- **総収益** -- 完了したすべてのレッスンからの総売上
- **レッスン数** -- 実施したセッション数
- **レッスンあたり平均** -- セッションあたりの平均収益

日付範囲ピッカーを使って、今週、今月、カスタム範囲、全期間の収益を確認できます。

### トップサービス

どのレッスンタイプが最も多くの収益を生み出しているかを確認：

- サービス名とタイプ
- 予約数
- サービスあたりの収益
- 総収益に占める割合

どのサービスがラーナーに最も響いているか、マーケティングのどこに注力すべきかを理解するのに役立ちます。

### トップ生徒

収益上位の生徒を確認：

- 予約レッスン合計数
- 生成された総収益
- 平均レッスン頻度

## 期間別収益

特定の期間を分析するために収益データをフィルタリングできます：

| ビュー | ユースケース |
|------|----------|
| **今週** | 今週のアクティビティを素早くチェック |
| **今月** | 月次の収益追跡 |
| **カスタム範囲** | 特定期間の比較や税務レポートの作成 |
| **全期間** | Loka利用開始からの全履歴 |

## 支払い方法の設定

Lokaは複数の支払い方法に対応しており、最適な方法で収益を受け取れます。

### Stripe Connect

Lokaは **Stripe Connect** を使った直接支払いに対応しています：

1. **設定 > 支払い** に移動
2. **Stripeに接続** をクリック
3. Stripeのオンボーディングプロセスを完了（本人確認、銀行口座の登録）
4. 接続後、レッスン料がStripeアカウントに直接振り込まれます

Stripe Connectでは、レッスン料の支払いごとに自動的に送金処理が行われます。Stripeダッシュボードで送金スケジュール（日次、週次、月次）を設定できます。

### PayPal

PayPalでの受け取りを設定するには：

1. **設定 > 支払い** に移動
2. PayPalのメールアドレスを入力
3. 設定されたスケジュールに従ってPayPalアカウントに送金されます

### Wise（TransferWise）

海外在住のエデュケーター向けに、**Wise** は競争力のある為替レートを提供しています：

1. **設定 > 支払い** に移動
2. Wiseアカウントの詳細を入力
3. 希望の通貨で送金されます

:::tip
複数の国のラーナーに指導している場合、国際送金ではWiseが最も良い為替レートと最低手数料を提供するのが一般的です。
:::

| **Loka Creator DFY** | $29/月 | DIYの全内容 + Done-For-You設定 + 月2時間のテクニカルサポート |

### 追加シート

- **エデュケーターシート**: シートあたり$6/月 -- スクール内の追加教師向け
- **ラーナーシート**: 生徒あたり$6/月 -- レッスン料の支払いに伴い自動更新

## 課金モデルの仕組み

| シートタイプ | 課金方法 | 自動キャンセル |
|-----------|-----------------|--------------|
| **エデュケーターシート** | 月額Stripeサブスクリプション | なし -- 手動キャンセルが必要 |
| **ラーナーシート** | ラーナーのレッスン料支払い時に更新 | あり -- レッスンが停止すると停止 |

アクティブな生徒数に応じてコストがスケールします。ラーナーが休止すると、シートコストも自動的に一時停止します。

## 税務と報告

Lokaは税務申告に必要なデータを提供します：

- マネジメントタブから収益履歴をCSVでエクスポート
- 税務期間に合わせて日付範囲でフィルタリング
- すべての金額は設定した通貨で表示されます

:::note
Lokaは源泉徴収や税務アドバイスを提供しません。独立エデュケーターとしての税務義務については、税理士に相談してください。
:::
