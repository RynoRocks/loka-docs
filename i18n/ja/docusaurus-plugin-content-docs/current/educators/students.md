# 生徒の管理

Lokaでは、スクール内のすべてのラーナーを一目で把握できます。進捗の追跡、レッスン履歴の確認、在籍管理を一か所で行えます。

## 生徒リスト

ダッシュボードの **マネジメント** タブから生徒名簿にアクセスできます。生徒リストには以下が表示されます：

- 生徒名とプロフィール写真
- 連絡先情報
- 在籍ステータス（アクティブ、非アクティブ、保留中）
- 受講完了レッスン数
- 最終レッスン日

検索バーで名前やメールアドレスで生徒を検索したり、フィルターでステータスやアクティビティレベルで絞り込んだりできます。

## 生徒プロフィール

生徒をクリックすると、詳細プロフィールが表示されます。各プロフィールには以下が含まれます：

### 連絡先情報

- 名前、メールアドレス、電話番号
- 希望言語とタイムゾーン
- 学習目標に関する記録メモ

### 学習履歴

学習の全記録：

- **セッションログ** -- 日付、所要時間、サービスタイプを含む全レッスン
- **Canvas履歴** -- メモ、キーワード、CIPを含むすべての[Living Canvas](/educators/canvas)セッション
- **ARP** -- 過去のセッションからの[Active Recall Pairs](/educators/canvas#active-recall-pairsarp)のライブラリ
- **進捗トレンド** -- 時間経過に伴うパフォーマンスの変化

### 予約履歴

過去および今後の予約を確認：

- レッスン日時
- サービスタイプと所要時間
- 支払い状況
- キャンセル履歴

## ラーナーのスクール参加方法

ラーナーがスクールに参加するにはいくつかの方法があります：

### 予約リンク

スクールの予約ページURLをラーナー候補に共有します。ラーナーは以下の手順で参加できます：

1. 予約ページにアクセス
2. 利用可能なサービスと空き時間を確認
3. Lokaアカウントを作成（またはログイン）
4. 最初のレッスンを予約し、支払いを完了

予約完了後、ラーナーは自動的にスクール名簿に追加されます。

### 直接招待

ダッシュボードから直接ラーナーを招待できます：

1. **マネジメント > 生徒** に移動
2. **生徒を招待** をクリック
3. 名前とメールアドレスを入力
4. アカウント設定リンク付きの招待メールがラーナーに届きます

### 管理者による割り当て

1つのスクールで複数のエデュケーターを管理している場合、管理者が管理画面から特定の教師に生徒を割り当てることができます。

## ラーナーシート

アクティブな生徒それぞれに **ラーナーシート** サブスクリプションが必要です。主な詳細：

- ラーナーシートは生徒あたり月額$6
- ラーナーがレッスン料を支払うと自動的に更新されます
- ラーナーがレッスンの予約を停止すると、シートサブスクリプションも停止します
- アクティブなシートがない生徒は、新しいレッスンの予約ができません

:::note
ラーナーシートはレッスン料の支払いに連動しています -- 手動で管理する必要はありません。ラーナーが予約・支払いをすればシートはアクティブのまま、予約が止まれば自然に期限切れになります。
:::

## 生徒記録の管理

### メモの追加

生徒プロフィールにプライベートメモを追加して、以下を記録できます：

- 学習目標とゴール
- 好みの指導スタイルやペース
- 今後のセッションで再度取り上げるべきトピック

これらのメモはエデュケーターのみに表示され、生徒には見えません。

### 進捗の確認

生徒のCanvas履歴とARPライブラリを使って進捗を評価できます：

- CIPトレンドを確認 -- 時間とともにエラーが減っているか？
- ARPの習熟度を確認 -- どのリコールペアが定着したか？
- セッション頻度を確認 -- 定期的に予約しているか？

### 生徒の非アクティブ化

ラーナーがアクティブでなくなった場合：

- プロフィールは記録保持のため名簿に残ります
- 過去のセッションデータ、Canvas、ARPは保持されます
- 新しいレッスンを予約することで、いつでも再アクティブ化できます
