# 進捗の追跡

Lokaは、これまでの成長と次に取り組むべきポイントを可視化します。レッスン、タスク、アセスメントの完了に伴い、進捗は自動的に記録されます。

## レベルシステム

言語レベルはエデュケーターによって設定され、段階的な進級体系に沿っています。レベルは通常、CEFR（A1〜C2）やスクール独自のレベルシステムに準拠しています。

現在のレベルはダッシュボードに表示され、以下を決定します：

- AI生成の復習教材（ARP）の難易度
- 推奨される語彙と文法ポイント
- グループレッスンがある場合のクラス分け

成長に伴い、エデュケーターがレベルを更新できます。レベル変更は記録されるため、学習の軌跡を振り返ることができます。

## 構造化カリキュラムレベル

一部のスクールでは、Lokaの構造化カリキュラム機能を使用して、学習を定義されたステージに整理しています。各レベルに含まれる内容：

- **学習目標** -- このレベルで達成すべきこと
- **必要レッスン数** -- 進級前に完了すべきレッスンまたはトピック数
- **アセスメント** -- 次のレベルへの準備状況を確認する定期チェック

カリキュラムの進捗状況は、ダッシュボードにプログレスバーとして表示されます。レベルのすべての要件を完了すると、エデュケーターが次のステージに進級させることができます。

## リワードポイント

以下のような積極的な学習行動でリワードポイントを獲得できます：

- レッスンの完了
- 期限内の宿題完了
- マイルストーンの達成（例：レッスン10回完了）
- 連続出席の継続

ポイントは累積され、ダッシュボードに表示されます。スクールによっては、ポイントを使って報酬や表彰を受けられる場合があります -- 詳細はエデュケーターに確認してください。

## クレジット

クレジットは、Lokaでレッスンを予約する際に使用する通貨です。スクールの設定により：

- **クレジット制予約** -- 各レッスンに一定数のクレジットが必要です。クレジットパックを購入または受領し、予約時にクレジットが差し引かれます。
- **サブスクリプション制予約** -- プランに一定数のレッスンが含まれます。プラン上限を超えるレッスンには、クレジットが使用される場合があります。

クレジット残高はダッシュボードで確認できます。クレジットが少なくなると、チャージのリマインダーが表示されます。

## バッジ

バッジは特定の達成やマイルストーンを表彰します。例：

- **初レッスン** -- Lokaで最初のレッスンを完了
- **ストリークマスター** -- 一定期間連続してレッスンに出席
- **ボキャブラリービルダー** -- 一定数のキーワードを復習
- **レベルアップ** -- 新しい熟練度レベルに進級

獲得したバッジはプロフィールとダッシュボードに表示されます。達成をひと目で確認できる、楽しい機能です。

:::tip
定期的にダッシュボードを確認して、進捗状況を把握しましょう。これまでの成長を見ることは、学習を続ける大きなモチベーションになります。
:::
