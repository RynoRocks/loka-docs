# レッスン教材と録音

レッスンが終わるたびに、Lokaはカバーされた内容の1ページサマリーを提供します。セッションを重ねるごとに成長する、パーソナルなテキストブックと考えてください。

## 1ページレッスンサマリー

完了したレッスンごとに、セッション中の内容をキャプチャしたサマリーページ -- **Canvas** -- が生成されます。含まれる内容：

### Canvasノート
レッスン中の教師によるメモ。主要なディスカッションポイント、説明、セッション中にホワイトボードに書かれた内容がカバーされます。

### キーワード＆語彙
レッスン中に導入された新しい単語やフレーズ。定義と例文付きで、レベルに応じてタグ付けされているため、文脈に沿って復習できます。

### ARP（Adaptive Review Phrases）
レッスン内容に基づくAI生成の復習フレーズです。ARPは、教材をさまざまな文脈や文型で提示することで、学んだ内容の練習を支援します。

### CIP（Correct & Incorrect Phrases）
レッスンからの添削内容で、実際の発言と改善版が並べて表示されます。各CIPは、なぜ修正が行われたかを理解できるよう、具体的な文法、語彙、または発音のポイントがハイライトされます。

### タスク
教師が割り当てた宿題やフォローアップタスク。ダッシュボードからタスクを完了済みにマークできます。

## レッスン後の教材へのアクセス

すべてのレッスン教材は、ダッシュボードの **マイレッスン** セクションから利用できます。以下が可能です：

- **日付で閲覧** -- レッスン履歴を時系列でスクロール
- **キーワードで検索** -- 過去のすべてのレッスンから特定のトピックや語彙を検索
- **教師で絞り込み** -- 特定の教師の教材のみを表示

教材は、教師がレッスンサマリーを完成させた直後に利用可能になります。新しい教材が準備されると通知が届きます。

## PDFダウンロード

オフライン学習や印刷のために、レッスンサマリーをPDFとしてダウンロードできます。

1. **マイレッスン** からレッスンサマリーを開く
2. **PDFをダウンロード** ボタンをクリック
3. PDFにはCanvasノート、キーワード、ARP、CIP、タスクが整ったフォーマットで含まれます

外出先での復習やスタディパートナーとの共有に便利です。

## MP3音声

Lokaは、テキスト読み上げ機能でレッスン内容のMP3音声ファイルを生成できます。以下に役立ちます：

- **リスニング練習** -- キーワードやフレーズの正しい発音を聞く
- **外出先での復習** -- 通勤中や運動中にレッスンサマリーを聞く
- **記憶の強化** -- 読むだけでなく聞くことで、記憶の定着が向上

MP3を生成するには：

1. レッスンサマリーを開く
2. **MP3** ボタンをクリック
3. 音声ファイルが生成され、ダウンロードまたはストリーミングが可能になります

:::tip
レッスンサマリーを定期的に見返しましょう。間隔反復 -- 時間を空けて繰り返し復習する方法 -- は、知識を長期記憶に定着させる最も効果的な方法のひとつです。
:::
