# アカウント設定

ダッシュボードの **アカウント設定** ページから、プロフィール、各種設定、セキュリティ設定を管理できます。

## プロフィール管理

プロフィールには、教師やスクールに表示される基本情報が含まれます。更新可能な項目：

- **表示名** -- 教師やレッスン記録に表示される名前
- **プロフィール写真** -- アカウントをパーソナライズする写真をアップロード
- **メールアドレス** -- ログインと通知に使用するアドレス。変更には再認証が必要
- **電話番号** -- オプション。スクールがSMSリマインダーを有効にしている場合に使用
- **自己紹介** -- 教師に表示される短い紹介文。興味や目標に合わせたレッスンの調整に役立ちます

プロフィールを更新するには：

1. 右上の名前またはアバターをクリック
2. **プロフィール** または **アカウント設定** を選択
3. 変更を加えて **保存** をクリック

## タイムゾーン設定

Lokaではすべてのレッスン時間がローカルタイムゾーンで表示されます。正確に設定することが、予約の混乱を防ぐために重要です。

- 初回サインアップ時にタイムゾーンは自動検出されます
- **アカウント設定 > タイムゾーン** で手動変更が可能です
- すべての予約確認、リマインダー、カレンダー登録は、選択したタイムゾーンが使用されます

:::caution
レッスン時間が正しく表示されない場合、最も多い原因はタイムゾーン設定の誤りです。サポートに問い合わせる前に、この設定を確認してください。
:::

## パスワードの変更

パスワードを変更するには：

1. **アカウント設定 > セキュリティ** に移動
2. 現在のパスワードを入力
3. 新しいパスワードを入力して確認
4. **パスワードを更新** をクリック

**パスワード要件：**
- 8文字以上
- 大文字、小文字、数字をそれぞれ1つ以上含む

パスワードを忘れた場合は、ログインページの **パスワードをお忘れですか？** リンクを使用してください。登録されたメールアドレスにリセットリンクが送信されます。

## 通知設定

Lokaは、レッスン、教材、アカウントのアクティビティに関する通知を送信します。受信する通知の種類を制御できます：

### メール通知

- **予約確認** -- レッスンの予約またはキャンセル時に送信
- **レッスンリマインダー** -- レッスン前に送信（例：24時間前と1時間前）
- **新教材の通知** -- 教師がレッスンサマリーを公開した際に送信
- **タスクリマインダー** -- 未完了の宿題がある場合に送信
- **アカウント更新** -- パスワード変更、メール変更、セキュリティアラート

### 設定の管理

1. **アカウント設定 > 通知** に移動
2. 各通知タイプのオン/オフを切り替え
3. 変更は即座に反映されます

:::tip
**レッスンリマインダー** と **新教材の通知** はオンのままにしておくことをお勧めします。タイムリーなリマインダーはレッスンの準備に役立ち、レッスン直後に教材を復習することで記憶の定着が大幅に向上します。
:::
