# レッスンの予約

Lokaでは、空き時間の確認、教師の選択、レッスンの予約がわずか数ステップで簡単に行えます。

## レッスンの予約方法

1. **サービスを選択** -- ダッシュボードから **レッスンを予約** をクリックし、レッスンタイプ（例：一般英語、ビジネス英語、試験対策）を選びます。利用可能なサービスは、スクールの提供内容に応じて異なります。

2. **日時を選択** -- カレンダーを参照して空き時間を確認します。空き時間はローカルタイムゾーンで表示されます。日、週、月のビューを切り替えられます。

3. **教師を選択** -- 選択した時間帯で対応可能な教師から選びます。各教師のプロフィールには、専門分野、対応言語、短い自己紹介が表示されます。

4. **受講形式を選択** -- レッスンの受講方法を選びます：
   - **オンライン** -- Zoomまたはスクール指定のビデオプラットフォーム経由
   - **対面** -- スクールが設定した物理的なロケーション
   - **ハイブリッド** -- 一部のスクールでは、同じレッスンで両方の形式を提供

5. **予約を確定** -- レッスンの詳細を確認し、**確定** をクリックします。オンラインレッスンの参加リンクを含む確認メールが届きます。

## 予約ルール

スクールによって、予約に関する特定のルールが設定されている場合があります。一般的な設定：

- **最低事前予約時間** -- 例：レッスンは最低24時間前までに予約が必要
- **最大事前予約期間** -- 例：2週間先まで予約可能
- **予約上限** -- 一度に保持できるアクティブな予約数を制限するプランもあります

スロットが利用できない場合は、すでに予約済みか、スクールの予約期間外の可能性があります。

## 予約の管理

ダッシュボードの **予約一覧** セクションから、今後のレッスンと過去のレッスンを確認できます。各予約には以下が表示されます：

- 日時と所要時間
- 教師名
- 受講形式（オンライン／対面）
- レッスン状況（予定、完了、キャンセル）

## 予約のキャンセル

レッスンをキャンセルするには、予約詳細を開いて **キャンセル** をクリックします。キャンセルポリシーはスクールによって異なりますが、一般的なルール：

- **無料キャンセル** -- レッスンの24時間以上前に行った場合
- **遅延キャンセル** -- 24時間以内の場合、完了レッスンとしてカウントされるか、ポリシーに応じて料金が発生する場合があります
- **無断欠席** -- 一般的に遅延キャンセルと同様に扱われます

キャンセルポリシーについて不明な場合は、スクールまたはエデュケーターに確認してください。

## グループレッスン

一部のスクールでは、複数のラーナーが同じセッションに参加するグループレッスンを提供しています。グループレッスンの予約時：

- 残り席数が表示されます
- 自分の席を個別に予約します -- 他のラーナーとの調整は不要です
- グループレッスンの教材はセッション後にすべての参加者に共有されます
- グループレッスンのキャンセルポリシーは、マンツーマンセッションとは異なる場合があります

:::tip
予約前に **アカウント設定** でタイムゾーンを設定して、すべての時間が正しく表示されるようにしてください。
:::
