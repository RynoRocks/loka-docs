# エデュケーターとラーナーの管理

Loka Creatorとして、スクール内の人材 -- 指導を行うエデュケーターと学習するラーナーの両方 -- を管理します。

## エデュケーターの追加

スクール内の各エデュケーターには **エデュケーターシート** が必要です（シートあたり$6/月）。Loka Creatorプランにはデフォルトでエデュケーターシート1席が含まれています。

新しいエデュケーターを追加するには：

1. **管理パネル > エデュケーター** に移動
2. **エデュケーターを追加** をクリック
3. 名前、メールアドレス、初期設定を入力
4. アカウント設定リンク付きの招待メールがエデュケーターに届きます

有効化後、エデュケーターは以下にアクセスできる専用ダッシュボードを取得します：

- レッスンスケジュールとLiving Textbook
- AI搭載のレッスンツール（ARP生成、アセスメント、フィードバック）
- 担当ラーナーの管理
- Canvasとホワイトボードツール

### エデュケーターの削除

エデュケーターを削除するには：

1. **管理パネル > エデュケーター** に移動
2. エデュケーターの名前をクリック
3. **エデュケーターを削除** を選択
4. 削除を確認

エデュケーターを削除すると、現在の請求期間の終了時にシートサブスクリプションがキャンセルされます。レッスン履歴と教材は保持されます。

## ラーナーアカウントの管理

ラーナーは招待またはサインアップリンクを通じてスクールに参加します。

### ラーナーの招待

1. **管理パネル > ラーナー** に移動
2. **ラーナーを追加** または **招待を送信** をクリック
3. ラーナーのメールアドレスを入力
4. アカウント作成の手順が記載されたメールがラーナーに届きます

### ラーナーシート

アクティブなラーナーそれぞれに **ラーナーシート** が必要です（$6/月）。エデュケーターシートとは異なり、ラーナーシートはレッスン料の支払い時に自動更新されます。ラーナーがレッスンの予約を停止すると、シートサブスクリプションは更新されません。

これにより、ラーナーシートを手動でキャンセルする必要はありません -- ラーナーが非アクティブになると自然に解約されます。

### ラーナー詳細の確認

ラーナー管理ページから確認できる項目：

- プロフィール情報と連絡先
- レッスン履歴と今後の予約
- 進捗データと現在のレベル
- シートサブスクリプション状況
- クレジット残高

## ユーザーステータス

Lokaのユーザーには以下のステータスがあります：

| ステータス | 意味 |
|--------|---------|
| **アクティブ** | アクティブなアカウントとアクティブなシートサブスクリプションを保持 |
| **保留中** | 招待済みだが、まだサインアップが完了していない |
| **非アクティブ** | シートサブスクリプションが失効またはキャンセルされた |
| **停止中** | Creatorによりアカウントが一時的に無効化された |

ステータスでユーザーリストをフィルタリングして、必要なユーザーをすばやく見つけることができます。

## 認証プロセス

新しいユーザーは認証プロセスを経ます：

1. **メール認証** -- ウェルカムメール内のリンクをクリックして、メールアドレスを確認
2. **プロフィール完成** -- 必要なプロフィール項目（名前、タイムゾーンなど）を入力
3. **シートの有効化** -- ユーザーのシートサブスクリプションが有効化（Creatorまたは支払いにより）

すべてのステップが完了すると、ユーザーのステータスが **アクティブ** に変更され、プラットフォームの全機能が利用可能になります。

:::tip
招待メールが届かなかった場合やリンクの有効期限が切れた場合は、ユーザーのプロフィールページから招待メールを再送信できます。
:::
