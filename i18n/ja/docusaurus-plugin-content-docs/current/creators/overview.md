# Loka Creatorの概要

**Loka Creator** とは、Lokaプラットフォーム上のスクールのオーナーまたは管理者です。Creatorとして、スクールの設定、エデュケーター、ラーナー、課金をすべてコントロールできます。

## Loka Creatorとは

Loka Creatorは、スクールを運営する人と考えてください。自分のビジネスを管理するソロの語学教師かもしれませんし、エデュケーターチームを統括するアカデミーオーナーかもしれません。いずれの場合も、Creatorの役割で、すべてを一か所から管理するツールが提供されます。

Loka Creatorプランに登録すると、以下が得られます：

- Lokaプラットフォーム上に自分専用のスクールインスタンス
- スクールのあらゆる側面を管理する管理パネル
- エデュケーターとラーナーを追加する機能
- 課金、設定、連携の管理権限

## 管理内容

Loka Creatorとして管理する項目：

### エデュケーター
- スクールへのエデュケーターの追加・削除
- 各エデュケーターには、レッスンツール、Living Textbook、AI機能、生徒管理を備えた専用ダッシュボードが付与されます
- エデュケーターシートの管理（各シートは別個のサブスクリプション）

### ラーナー
- ラーナーをスクールに招待
- ラーナーの進捗、レッスン履歴、アカウント状況の確認
- ラーナーシートの課金管理

### 課金＆サブスクリプション
- Loka Creatorプラン（DIY、DFY、Aspire）の選択・管理
- エデュケーターシートとラーナーシートの追加購入
- 請求書と支払い方法のStripe課金ポータルへのアクセス
- スクール全体のシート利用状況を監視

### プラットフォーム設定
- 予約ルール、タイムゾーン、通貨の設定
- 連携の設定（Stripe、Zoom、Asanaなど）
- メールテンプレートと通知のカスタマイズ
- ホワイトラベルオプションの設定

## 管理パネルへのアクセス

ログイン後、ナビゲーションメニューに **管理パネル** が表示されます。管理パネルでアクセスできる項目：

- **ダッシュボード** -- アクティブなラーナー、今後のレッスン、最近の予約など、スクールのアクティビティの概要
- **エデュケーター** -- 指導スタッフとその権限の管理
- **ラーナー** -- すべてのラーナーアカウントの閲覧・管理
- **課金** -- サブスクリプション、シート、支払い設定の管理
- **設定** -- スクール全体の設定と連携の管理

:::tip
自分自身もレッスンを行う場合は、ナビゲーションバーのロール切り替えで、Creator（管理者）ビューとエデュケーター（指導）ビューを切り替えできます。
:::
