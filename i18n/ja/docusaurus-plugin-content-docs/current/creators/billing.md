# 課金とサブスクリプション

Lokaはシートベースのサブスクリプションモデルを採用しています。スクール向けのCreatorプランを選び、スクールの成長に合わせてエデュケーターシートとラーナーシートを追加します。

## Loka Creatorプラン

| プラン | 料金 | 内容 |
|------|-------|-------------|
| **Loka Creator DIY** | $6/月 | エデュケーターシート1席 + ラーナーテストシート1席。セルフサービスのセットアップで、すべて自分で設定。 |
| **Loka Creator DFY** | $29/月 | DIYの全内容 + Done-For-Youのプラットフォーム設定 + 月2時間のテクニカルサポート。 |
| **Loka Creator Aspire** | $199/月 | DFYの全内容 + 月2時間のファウンダーコーチングで教育ビジネスを成長。 |

すべてのプランに、Lokaプラットフォームの全機能が含まれます：Living Textbook、AIツール、予約システム、進捗追跡、各種連携。

### 14-Day Challenge

Lokaが合っているか分からない場合は、**14-Day Challenge**（$39 一回払い）がおすすめです。14日間、ガイド付きでプラットフォームのすべての機能を体験できます。その後、月額プランにアップグレードできます。

## エデュケーターシート

スクール内の各エデュケーターには専用のシートが必要です。

- **料金：** エデュケーターシートあたり$6/月
- **課金方法：** 標準の月額Stripeサブスクリプション
- **含まれるシート：** Creatorプランにはエデュケーターシート1席が含まれます
- **追加方法：** **管理パネル > 課金 > エデュケーターシートを追加** から追加シートを購入
- **キャンセル：** 手動 -- 不要になったエデュケーターシートを自分でキャンセル

## ラーナーシート

アクティブなラーナーにもシートが必要ですが、ラーナーの課金は仕組みが異なります。

- **料金：** ラーナーシートあたり$6/月
- **課金方法：** ラーナーがレッスン料を支払うと自動更新
- **自動キャンセル：** ラーナーがレッスン料の支払いを停止すると、シートは更新されません
- **手動管理不要：** ラーナーシートはレッスン料の支払いに連動して有効化・無効化されます

このモデルにより、非アクティブなラーナーに対して料金が発生することはありません。シートはアクティブな生徒数に応じて自然にスケールします。

## Stripe課金ポータル

すべての支払いはStripeを通じて処理されます。**管理パネル > 課金 > 課金管理** からStripe課金ポータルにアクセスできます。

ポータルでできること：

- 請求書の閲覧とダウンロード
- 支払い方法（クレジットカードなど）の更新
- 現在のサブスクリプションと更新日の確認
- 支払い履歴の閲覧

## サブスクリプションの管理

### プランのアップグレード

より上位のCreatorプランに変更するには：

1. **管理パネル > 課金** に移動
2. **プラン変更** をクリック
3. 新しいプランを選択
4. 変更は即座に適用されます。現在の請求期間の残り分について、差額が按分課金されます。

### プランのダウングレード

ダウングレードは現在の請求期間の終了時に適用されます。含まれる機能が少ないプランにダウングレードする場合は、切り替え前にセットアップの互換性を確認してください。

### プランのキャンセル

Loka Creatorサブスクリプションをキャンセルするには：

1. **管理パネル > 課金** に移動
2. **サブスクリプションをキャンセル** をクリック
3. 現在の請求期間の終了までアクセスが継続されます
4. キャンセル後、データは90日間保持されます

:::caution
Creatorプランをキャンセルすると、スクールに関連するすべてのエデュケーターシートとラーナーシートもキャンセルされます。保持期間が終了する前に、必要なデータをエクスポートしてください。
:::

### シートの追加・削除

- **シートの追加：** **管理パネル > 課金 > エデュケーターシートを追加**（ラーナーは追加するだけで、支払い時に自動的にシートが有効化されます）
- **エデュケーターシートの削除：** エデュケーターのプロフィールから **エデュケーターを削除** をクリック。シートは請求期間の終了時にキャンセルされます。
- **ラーナーシートの削除：** ラーナーがレッスン料の支払いを停止すると自動的にキャンセルされます。手動操作は不要です。
